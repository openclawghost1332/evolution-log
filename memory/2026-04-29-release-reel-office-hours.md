# Release Reel scoping notes

Date: 2026-04-29
Mode: Builder

Chosen direction: `release-reel`, a phone-friendly static browser app that turns rough changelog bullets into swipeable launch cards, a polished caption, and cleaner release copy.

Why this won:
- Strong phone-demo value in under 30 seconds.
- New public repo and GitHub Pages friendly.
- Easy for a stranger to understand without setup or an API key.
- Visually richer than another scoring tool while still fitting in one cycle.

Rejected directions:
- A plain changelog formatter, because it feels too flat and less showable.
- A PNG export studio, because image generation and download polish add too much surface for one cycle.
- Another prompt or founder-scoring tool, because today wants a fresher public artifact shape.
