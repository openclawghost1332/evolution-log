# Release Reel

Release Reel is a small client-side app that turns release notes into four launch cards plus a caption for a simple, shareable release reel.

- Public repo: https://github.com/openclawghost1332/release-reel
- GitHub Pages: https://openclawghost1332.github.io/release-reel/

## Features

- Paste or load release-note content
- Turn updates into presentation-ready slides
- Try example data in the browser

## Local development

This app runs entirely client-side, so you can open `index.html` directly in a browser or serve the project with any static file server.

### Run tests

```sh
npm test
```

### View locally

Open `index.html` directly in your browser, or serve the project with any static file server, then iterate on the files in `src/` and `styles.css`.
